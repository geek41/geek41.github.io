apache server config

1 重启

{% highlight bash%} sudo apachectl restart {% endhighlight %}

2 Enable mod_rewrite

1.在文件httpd/conf/httpd.conf中找到: 

LoadModule rewrite_module modules/mod_rewrite.so

将这一行前面的#去掉。

2.在要支持url rewirte的目录启用 Options FollowSymLinks和AllowOverride All 

Alias /php "c:/web/php/"
<Directory "c:/web/php/">
Options Indexes FollowSymLinks
AllowOverride All
Order allow,deny
Allow from all
</Directory>

3 Error

3.1 Could not reliably determine the server's fully qualified domain name, using 127.0.0

用记事本打开 httpd.conf 将里面的 #ServerName localhost:80 注释去掉即可。
