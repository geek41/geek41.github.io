--- layout: view comments: true title: Jquery datepicker z-index categories: web前端 tags: [javascript,datepicker] date: 2013-8-3 keywords: javascript,datepicker description: jquery ui-datepicker z-index 显示问题 ---

场景:有时在页面弹出层的界面调用jquery-ui的datepicker,控件显示在了遮照层下面。这是因为jquery-ui datepicker的源码实现为: 

$(input).zIndex() + 1

解决方法1: 

$("#txtFillInDate").datepicker({
monthNames: ['一月', '二月', '三月', '四月', '五月', '六月', '七月', '八月', '九月', '十月', '十一月', '十二月'],
dayNamesMin: ['日', '一', '二', '三', '四', '五', '六'],
dateFormat: 'yy-mm-dd',
beforeShow: function () {
setTimeout(
function () {
$('#ui-datepicker-div').css("z-index", 15);
}, 100
);
}
});
