用wget保存网页，文档，api等，方便快捷。例如Spring Framework Reference Documentation.

1 用wget保存网页，文档，api等

{% highlight bash %} wget -mxnp -N http//docs.spring.io/spring/docs/4.2.0.BUILD-SNAPSHOT/spring-framework-reference/htmlsingle/ {% endhighlight %}

-m wget会自动判断合适的参数来制作镜像站点。此时，wget会登录到服务器上，读入robots.txt并按robots.txt的规定来执行

-x –force-directories 强制建立服务器上一模一样的目录

-np –no-parent 仅下载html/文件夹下的网页。

-N –timestamping 只下载比本地新的文件

-nc, –no-clobber 不要覆盖已经存在的文件
